# 0G DataBounty Submission Pack

## Basic Project Information

**Project name:** 0G DataBounty

**One-sentence description:** A verifiable dataset bounty market where AI teams fund data needs and validator agents score submissions using 0G Storage, Compute, and Chain.

**Short summary:** 0G DataBounty lets AI builders create escrowed dataset bounties, contributors submit dataset manifests stored on 0G Storage, and validator agents score submissions with 0G Compute. The registry contract on 0G Chain records bounty, submission, validation, and settlement events so judges can verify activity directly.

## 0G Components Used

- **0G Chain:** native 0G escrow and dataset bounty registry.
- **0G Storage:** bounty metadata, dataset manifests, and validation reports.
- **0G Compute:** validator-agent scoring through mainnet inference services.
- **Agent ID:** validator agents are represented by wallet-backed agent profiles.
- **Privacy / Secure Execution:** manifests and samples can be validated without exposing sensitive raw datasets; private payloads should be encrypted before Storage upload.
